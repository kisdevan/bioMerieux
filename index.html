① 고객 문의 자동화 — 상세

목표

  • 응답 시간 단축, 누락·중복 대응 최소화
  • FAQ 축적 및 템플릿 품질 향상

흐름(Flow)

  • Outlook 수신 → Power Automate 트리거
  • LLM 요약·분류(환불/배송/기술지원)
  • 템플릿 초안 생성 → 승인 규칙 → 발송

샘플 프롬프트

  • "다음 메일을 요약하고 카테고리로 분류해줘. 또한 표준 답변 초안을 3문장으로 작성"

지표(KPI)

  • 평균 응답시간, 1차 해결률, 이탈/재문의율

툴 스택

  • Outlook · Power Automate · Copilot/GPT · SharePoint

리스크·컴플라이언스

  • 민감정보 마스킹, 승인 단계 의무화, 감사 로깅

⬅ 시나리오 인덱스로 옵션 비교로

② 보고서 자동 생성 — 상세

목표

  • 보고서 작성 시간 80% 절감, 포맷 표준화

흐름(Flow)

  • Excel/DB 추출 → GPT 요약/서술 → Word 템플릿 병합
  • PDF 변환 → 결재 라우팅 → 자동 발송

샘플 프롬프트

  • "아래 표 데이터를 경영 요약 5문장과 핵심 지표 표로 작성"

지표(KPI)

  • TAT, 오류율, 재작업률

툴 스택

  • Excel · Power Automate · Word/SharePoint · GPT

리스크·컴플라이언스

  • 버전관리, 문서 분류 보안, 검토자 서명 캡처

⬅ 시나리오 인덱스로 옵션 비교로

③ 회의록 요약 & 액션 아이템 — 상세

목표

  • 회의 → 실행 전환 속도 향상, 누락 방지

흐름(Flow)

  • Teams 자막 캡처 → GPT 요약
  • Action Item 추출 → Planner/Trello 등록

샘플 프롬프트

  • "회의록에서 결정/보류/담당/마감일을 표로 추출"

지표(KPI)

  • 액션 완료율, 평균 처리기간, 재오픈율

툴 스택

  • Teams · Power Automate · Planner/Trello · GPT

리스크·컴플라이언스

  • 회의 참가자 동의, 민감 발언 비공개 처리

⬅ 시나리오 인덱스로 옵션 비교로

④ 다국어 번역 & 글로벌 협업 — 상세

목표

  • 번역 품질·속도 향상, 현지화 톤 일관성

흐름(Flow)

⬅ 시나리오 인덱스로 옵션 비교로
